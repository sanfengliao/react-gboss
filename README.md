## React项目 gboss
* gboss是一个仿招聘前后端web应用, 前端使用的技术为react + redux + axios+atd-design, 后端使用express + mongodb, 前后端交互使用cors(不知道为啥配置代理不起作用，所以只能使用跨域了), 使用socket.io做及时聊天功能
###使用
```
# 安装依赖
npm install
# 启动服务端
npm run server
# 启动客户端
npm start

```
